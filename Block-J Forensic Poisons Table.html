Block-J Forensic Medicine Poisons Table

Poison / Toxin Class Key Effect Classical Sign/Symptom Fatal Dose Antidote
Morphine / Opium Somniferous (Cerebral) CNS depression → respiratory depression Pinpoint pupils (miosis) + slow pulse + slow respiration 200-300 mg Naloxone
Heroin Somniferous (Cerebral) CNS depression (same as morphine) Pinpoint pupils + respiratory depression Variable (addicts tolerant) Naloxone
Datura / Atropine Deliriant (Cerebral) Anticholinergic – blocks muscarinic receptors Hot, dry, red skin + dilated pupils + delirium + urinary retention 100-150 mg (atropine) Physostigmine
Cannabis Deliriant (Cerebral) CB1/CB2 receptor agonist Euphoria + red eyes + increased appetite No fatal dose Supportive only
Strychnine Spinal Blocks spinal inhibition → increased reflex excitability Opisthotonos (arched back) + convulsions triggered by minimal stimuli Adults 40-100 mg / Children 15 mg Diazepam + airway support
Snake Venom (Neurotoxic) Peripheral Blocks neuromuscular transmission Descending flaccid paralysis + ptosis + diplopia Species dependent Polyvalent antivenom
Botulinum Toxin Peripheral Blocks ACh release at presynaptic terminal Descending symmetrical flaccid paralysis + cranial nerve involvement Extremely small (nanograms) Botulinum antitoxin
Curare Peripheral Competitive antagonist at nicotinic receptors Flaccid paralysis + respiratory failure Variable Neostigmine + respiratory support
